# PreçoMapa Mobile (Expo)

App mobile mockado para iOS/Android com 5 telas core:

- Home
- Buscar
- Mapa (mock)
- Favoritos
- Alertas

## Rodar localmente

```bash
npm --prefix mobile install
npm --prefix mobile run start
```

## Checks

```bash
npm --prefix mobile run lint
npm --prefix mobile run typecheck
```

> Observação: este diretório depende de pacotes Expo que precisam ser baixados da internet.
